# Changelog

## [0.2.3] - 2026-07-17

### Added

- 保存每次成功抓取的本地样本摘要，并展示详情、薪资、技能、经验和学历覆盖率。
- 支持最近两次跨日、同条件抓取批次比较，明确标注小样本、缺失数据和不可比较原因。
- 将当前市场样本上下文接入主简历 AI 助手，保留事实确认、逐项审核和版本回滚流程。

### Changed

- 岗位报告改用“当前本地有限样本”的保守表述，移除 7/30 天趋势和未经支持的市场推断。
- 抓取完成后复用已保存岗位生成报告摘要，不增加 BOSS 访问频率或后台抓取。
- 优化岗位分析决策面板与主简历工作台的交互和可审核修改流程。

## [0.2.2] - 2026-07-15

### Added

- BOSS 直聘完整城市搜索、热门城市快捷选择与键盘操作。
- 自动保存并恢复最近一次完整岗位搜索条件。
- 关于页面的 GitHub Issues 反馈入口与本地化更新时间展示。

### Changed

- 强化岗位详情正文提取、登录失效识别与后台标签页兼容性。
- 应用 identifier 更新为 `io.github.aijobapp`，并保留旧数据迁移支持。
- Release Notes 仅使用当前版本的更新日志内容，并增加对应校验。

## [0.2.1] - 2026-07-15

### Changed

- GitHub Release 仅发布三平台安装/更新包与 `latest.json`。
- Release 标题统一使用版本号。

## [0.2.0] - 2026-07-14

### Added

- Tauri 签名更新包、每日更新检查和用户确认安装流程。
- SQLite 自动/手动备份、恢复、分项数据清除和旧 identifier 数据迁移。
- 首次启动隐私确认、关于与诊断信息、脱敏诊断包。
- Windows 10/11 x64 与 macOS 12+ 的公开 Beta 发布配置。

### Changed

- 应用 identifier 更新为 `io.github.kylekw.aijobapp`。
- Windows 公开安装器统一为 current-user NSIS。
- Python sidecar 使用锁定的 uv 依赖和固定工具链构建。

### Security

- 发布产物增加 updater 签名、SHA-256 校验和、SBOM 和全历史密钥扫描门禁。
